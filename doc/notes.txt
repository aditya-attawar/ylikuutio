How to export UV layout in Blender:
* in UV/Image Editor mode -> UVs -> Export UV Layout , export as a .png file.

Load .png in GIMP.

Saving file as BMP (it needs to be a 24-bit BMP):
-> Export
-> Windows BMP Image (*.bmp)
-> Compatibility Options
-> Do not write color space information (raksi ruutuun)
-> Advanced Options
-> 24 bits (R8 G8 B8)
-> Export
