UV layoutin exporttaus Blenderissä:
* UV/Image Editor -moodissa -> UVs -> Export UV Layout , .png-tiedostona ulos.

Ladataan .png GIMP:ssä.

Ladataan toistaiseksi funktiolla `GLuint load_BMP_texture(std::string filename)`:

Tiedoston tallennus `*.bmp`-formaatissa (toistaiseksi pitää olla 24-bittinen BMP):
* -> Export
  -> Windows BMP Image (*.bmp)
  -> Compatibility Options
  -> Do not write color space information (raksi ruutuun)
  -> Advanced Options
  -> 24 bits (R8 G8 B8)
  -> Export
